# 🗺️ 머니맵 (MoneyMap)

엑셀 매크로로 쓰던 가족 자산관리 시스템을 **설치·서버·인터넷 없이** 브라우저에서 바로 쓰는 **단일 파일 웹앱**으로 옮긴 버전입니다. 현재 **v1.10.0**.

🔗 **바로 쓰기**: https://prjessy.github.io/finance/

## 사용법
1. `index.html` 더블클릭(로컬) 또는 위 링크 접속
2. (`xlsx.full.min.js`는 같은 폴더에 함께 있어야 엑셀 기능 동작)
3. **🎲 샘플 데이터**로 미리보기 → **🗑 초기화** 후 실제 데이터 입력

## 주요 기능
- **자산 6종**: 동산(예·적금)·부동산·투자자산·공모주(IPO)·연금/보험·대출
- **가계부**: 현금흐름, 월별 현황표, 반복거래 자동생성, 지출 자동분류
- **분석**: 대시보드(차트·순자산 추이), 월간 리포트(전월/전년 대비·저축률·구독·기부금), 금융 캘린더+현금흐름 예측, 피벗분석, 통합검색
- **계획**: 목표·예산, 일일 사용가능액, 자산배분 목표, 저축목표(여러개), 연말정산 세액공제
- **투자**: 수익률·연환산(CAGR), 공모주 청약 캘린더
- **시스템**: 엑셀 양방향(가져오기/내보내기/탭별), 날짜별 백업, 클라우드 동기화(Google 로그인·선택), 만기 알림, 다크모드, 모바일 대응, 인쇄/PDF

## 데이터 & 프라이버시
- **로그인 안 함** = 데이터가 그 브라우저(localStorage)에만 저장, 아무도 못 봄. 엑셀로 백업.
- **Google 로그인** = Firebase 클라우드에 저장 → 여러 기기 동기화. 사용자별 데이터 격리(서로 못 봄).
- 실제 금융 데이터·스크린샷은 `.gitignore`로 제외(공개 레포 안전).
