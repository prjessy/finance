# 자산관리 시스템 (웹 버전)

엑셀 매크로로 쓰던 개인 자산관리 시스템을 **설치·서버·인터넷 없이** 브라우저에서 바로 쓰는 단일 파일 웹앱으로 옮긴 버전입니다.

## 사용법
1. `자산관리시스템.html` 파일을 더블클릭 → 브라우저에서 열림
2. (`xlsx.full.min.js` 는 같은 폴더에 함께 있어야 엑셀 기능 동작)
3. **🎲 샘플 데이터** 로 미리보기 → **🗑 초기화** 후 실제 데이터 입력

## 탭 구성
- **요약** — 순자산/총자산/부채, 자산 구성 그래프, 이번 달 현금흐름, 만기 알림
- **현금흐름_현황표** — 월별 수입·지출·누적 자동 집계
- **동산 입력창** — 예·적금 신규/수정 (두 패널)
- **현금흐름** — 수입/지출 내역
- **동산 / 부동산 / 투자자산 / 연금보험** — 자산별 목록·합계·손익
- **동산만기** — 만기일·예상이자·세후 만기금액 자동 계산 (D-day)

## 기능
- 📥 **엑셀 가져오기** — 시트명(동산/부동산/투자자산/연금보험/현금흐름) 자동 매칭, 헤더 별칭 인식
- 📤 **엑셀 내보내기** — 만기 계산 시트 포함 .xlsx 저장
- 데이터는 브라우저 localStorage 에 자동 저장 (외부 전송 없음)

## 주의
실제 자산 데이터(엑셀 파일)와 스크린샷은 `.gitignore` 로 제외됩니다. 공개 레포에 개인 금융정보를 올리지 마세요.
