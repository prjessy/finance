# 자산관리 시스템 — 기능 아이디어 / 로드맵

> 앞으로 추가할 만한 기능 모음. **구현 전 아이디어 단계**이며, 필요할 때 골라서 개발한다.
> 모든 "실용/부가" 기능은 별도 서버 없이 브라우저 JS만으로 구현 가능. (AI 코치 제외)

---

## ✅ 이미 구현된 기능 (v1.7.0 기준)

- 자산 관리: 동산(예·적금) / 부동산 / 투자자산 / 연금·보험 / 대출·부채
- 현금흐름 + 월별 현황표 + 반복거래(자동 생성)
- 용돈관리 (사람별 월 용돈 설정·지급·기록)
- 요약 대시보드 (도넛/막대/순자산 추이 그래프, 만기 알림 배너)
- 목표·예산 (순자산 목표, 분류별 예산, 자산배분 목표)
- 월간 리포트 (전월 대비, 저축률, 지출 Top, 구독·정기결제 추적)
- 금융 캘린더 + 현금흐름 예측
- 동산만기 자동 계산 (세후), 피벗분석
- 지출 자동분류 규칙
- 엑셀 가져오기/내보내기(탭별·전체), 날짜별 백업, 초기화 전 자동 백업
- Firebase 클라우드 동기화 + Google 로그인 (선택), 로컬 전용 사용 가능
- 다크모드, 모바일 최적화, 인쇄/PDF, 버전·릴리즈 노트

## 메모
- **실용 우선 1~6번**이 투자 대비 효과가 가장 큼.

- AI/자동연동은 서버·비용·인증 장벽이 커서 당장은 보류.
