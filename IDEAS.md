## ✅ 완료 (구현됨)

### 자산 관리
- [x] 동산(예·적금) — 입력창(신규/수정 두 패널), 목록, 만기 자동계산(세후)
- [x] 부동산 — 취득가/현재가/대출/순자산
- [x] 투자자산 — 종목, 평가/손익, **수익률·연환산(CAGR)**, 종류 세분(국내 일반주/공모주 등)
- [x] 공모주(IPO) — 종목·주관사·청약일·환불일·상장일·공모가, **청약 캘린더**
- [x] 연금/보험 — 납입현황, 예상수령액
- [x] 대출·부채 — 신용/담보/마이너스 등, 잔액이 순자산에서 차감

### 가계부
- [x] 현금흐름 — 수입/지출 기록, 분류(구독·용돈·기부/헌금·경조사 포함)
- [x] 현금흐름_현황표 — 월별 집계·누적
- [x] 반복거래 — 월급·월세·구독 자동 생성
- [x] 지출 자동분류 규칙 — 키워드→분류 (엑셀 가져올 때 자동)

### 분석·계획
- [x] 대시보드(요약) — 순자산/자산구성 도넛/월별 현금흐름 막대/**순자산 추이 선그래프**/만기 배너
- [x] 월간 리포트 — 전월 대비, **전년 동월 대비**, 저축률, 지출 Top, **구독·정기결제**, **올해 기부금 합계**
- [x] 금융 캘린더 + 현금흐름 예측(6개월)
- [x] 목표·예산 — 순자산 목표, 분류별 예산, **일일 사용가능액**, 자산배분 목표, **여러 저축목표**, 연말정산 세액공제(연금저축·IRP)
- [x] 피벗분석 — 행/열/값/집계
- [x] 통합검색 — 전체 탭 한 번에

### 시스템
- [x] 엑셀 양방향 — 전체 가져오기/내보내기, 탭별 업로드·다운로드·양식

- [x] 버전/릴리즈 노트

## ⏳ 미완료 (제안했으나 미착수 — 무료·구현 가능)

| 항목 | 성격 | 난이도 | 메모 |
|---|---|:---:|---|
| 모바일 헤더 "⋯ 더보기" 메뉴 | 폴리시 | 하 | 폰에서 헤더 버튼 8개가 여러 줄로 접힘 → 정리 필요 |
| 코인 시세 자동갱신 | 자동화 | 하 | 업비트 공개 API(CORS 허용)로 프록시 없이 가능 |
| PWA(폰 홈화면 설치) | 폴리시 | 중 | manifest + service worker |
| 고정 vs 변동 지출 분석 | 분석 | 하 | 분류를 고정/변동으로 묶어 비중 |
| 목표 도달 예상 시점 | 분석 | 하 | 저축률 기준 "목표까지 N개월" |
| 은행/카드 CSV 매핑 | 편의 | 중 | 카드사별 CSV 양식 자동 인식 |
| 기부금/구독 등 연간 리포트 확장 | 분석 | 하 | 이미 합계는 있음, 상세화 |

## 🔮 향후 과제 (서버·인프라·구조 변경 필요 — 보류)

| 항목 | 왜 보류 |
|---|---|
| 미국주식 시세 자동갱신 | Finnhub 무료키 + Cloudflare Worker 프록시 필요 |
| 국내주식(KRX) 시세 자동 | 공식 무료 API 약함, 비공식은 불안정 |
| 계좌 자동연동(마이데이터) | 금융사 제휴·인증 필요 (개인앱 불가) |
| AI 재무코치 | Claude API 키(비밀) + 프록시 서버 + 토큰 비용 |
| 영수증 OCR 자동입력 | OCR 라이브러리/서비스 필요 |
| 가족 공유 가계부(같은 데이터 함께 보기) | Firestore 문서 구조 변경(현재는 사용자별 격리) |
| 동시편집 충돌 방지 | 단일 문서 last-write-wins → 머지 로직 필요 |
